# Sloaner 翻译器

一个简单易用的在线翻译工具，支持中文、英语和日语之间的互译。

## 功能特点

- 简洁的用户界面
- 支持多种语言互译
- 响应式设计，适配各种设备
- 使用 MyMemory Translation API，无需配置
- 完全免费使用，每日有次数限制1000次

## 本地运行

点开index.html即可使用

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- MyMemory Translation API

## API 使用限制

- 每天免费翻译额度：1000个单词
- 无需 API 密钥
- 建议在生产环境中注册 MyMemory 账户以获得更多配额

## 项目结构

- index.html：主页面
- styles.css：样式文件
- script.js：脚本文件



## 注意事项

1. 请遵守 MyMemory API 的使用条款
2. 如果需要更大的翻译量，建议：
   - 注册 MyMemory 账户获取更多配额
   - 或迁移到其他翻译服务（如 Google Cloud Translation API）
